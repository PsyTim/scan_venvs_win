
## [0.2.1] - 2026-07-03

## [0.2.0] - 2026-07-03
### Добавлено
- Кнопки сортировки результатов по: размеру, дате создания, дате изменения, пути
- После завершения сканирования результаты отображаются с возможностью переключения сортировки
- Ожидание завершения вычисления размеров перед отображением отсортированных результатов
- Автоматическая сортировка найденных папок `.venv` по дате создания (новые сверху) после завершения сканирования
- По клику на строке с найденной папкой открывается проводник Windows в родительской папке
- Отображение общего размера всех найденных папок в реальном времени с обновлением по мере подсчёта
- Фоновый подсчёт размера найденных папок `.venv` с отображением в реальном времени
- Обновление строки с результатом после завершения подсчёта размера
- Форматирование размера в удобочитаемом виде (Б, КБ, МБ, ГБ, ТБ)

## [0.1.4] - 2026-07-03
### Добавлено
- Иерархический прогресс-бар: каждая папка первого уровня даёт вклад 1/N, внутри неё папки второго уровня делят этот вклад
- Отображение текущей сканируемой папки второго уровня в реальном времени
- Сбор папок первого и второго уровней для точного расчёта прогресса
- Устранено двойное сканирование папок

### Изменено
- Полностью переработана логика сканирования: теперь используется os.walk для каждой папки первого уровня, прогресс обновляется при входе в папки второго уровня
